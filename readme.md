# Wordly discovery web application using wordlysparrow web api
well wordly sparrow is a responsive web app less versatile in nature, this web app enables user to seearch for a word get its pronounciation and save favourites in the fav tab space
it utilises the free dictionary api 

## you can view the live demo here
https://wordly-project.vercel.app/

### features that sparrowanime have
-search: enables you to search for a word return the meaning and get the pronounciation
-theme: has a dark theme: toggles between dark and light them customized by the linear gradient color
-UI/UX: has a good user design interface with customized font awesome features
-add to favouritebutton: provides a feature to add to favourties in the down container below
-provides an error feedback mechanism displayed incase of server api crush or word not found
-provides local storage of searched word andfavourite word
